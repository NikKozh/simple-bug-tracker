# simple-bug-tracker
The simpliest bug tracker with adding and deleting tasks. Scala + Play Framework, using PostgreSQL and Slick as ORM. Apache Lucene for full-text search.

To use it you need:
* PostgreSQL Server
  * address: localhost 
  * user: "postgres" 
  * password "1234"
* sbt run

The server will be on http://localhost:9000 address.
